Lion Rock Trail Map

Visit Website View Code on GitHub
Google Maps JavaScript API Google Directions API HTML CSS GitHub

The website displays Lion Rock Trail in Hong Kong using Google Maps API. Google Maps JavaScript API and Google Directions API were used to render and customize the points of interest and the hiking route. The web map was embedded into the website, allowing different user interactions such as checking the elevation of the specific point of interest through info windows.

BC Wildfire Dashboard

Visit Dashboard
ArcGIS Map Viewer ArcGIS Dashboards

The interactive dashboard displays wildfire location points in British Columbia for the current fire season through ArcGIS Dashboards. The wildfire location points, which are updated from the operational systems every two hours, and the jurisdiction boundaries of the BC Wildfire Fire Centres, were obtained from the BC's Map Hub. The dashboard was designed strategically to display the spatial information and the relevant information on a single screen effectively, with the goal to help the users to make informed decisions.
